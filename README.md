# Chave 10 — Sistema de Gestão para Oficinas Mecânicas

## Estrutura do projeto

```
chave10-novo/
├── backend/    Node.js + Express + SQLite
└── frontend/   React + Vite

demo/           Versão demo HTML puro (localStorage, sem backend)
```

## Como rodar o sistema

**Backend:**
```bash
cd chave10-novo/backend
npm install
npm start
```

**Frontend:**
```bash
cd chave10-novo/frontend
npm install
npm run dev
```

Acesse: http://localhost:5173

## Demo
Abra `demo/index.html` diretamente no browser. Não precisa de servidor.
